# 홀덤메이트 로드맵
기능 백로그 + 스펙 인덱스. 세부 내용은 각 `specs/*.md` 파일에서 관리하고, 여긴 전체 상태만 한눈에 본다.

## 스펙 목록
| 파일 | 상태 | 한 줄 요약 |
|---|---|---|
| home-grid-redesign.md | 초안 | 홈 카드 그리드 전환 + 유저영역(마스터/로그인) 우측 상단 이동 |

## 백로그 (아직 스펙 파일 없음)
- 스트럭쳐 자동 설계 도우미 — 초안
- 홈 화면 최근 게임 카드 노출 — 미정, 논의 중
- 계정별 권한 시스템 (마스터의 계정 등록 권한, 참여자 본인 게임만 목록 노출) — 미정, 규모 큼 (Firestore 스키마 변경 + 보안 규칙 재설계 선행 필요)
- Push/Fold 차트 팔레트를 홈 화면과 통일 — 미정, 논의 중

## 관리 규칙
- 스펙 파일은 기능 단위로 하나씩 만든다. "확정 → 구현 → 완료" 사이클이 끝난 스펙에 큰 요구사항이 새로 붙으면, 같은 파일을 계속 늘리지 말고 새 파일로 분리한다.
- 아직 "초안" 상태에서 협의 중인 스펙은 새 파일을 만들지 않고 기존 파일을 계속 수정한다.
- 스펙이 "완료"되어도 파일은 삭제하지 않고 보존한다 (추적 기록 목적). 이 표의 상태만 갱신한다.
- 새 스펙 파일이 생기거나 상태가 바뀌면 이 표를 함께 갱신한다.
